# Data source

Analysis data and author code are **not committed**. They are fetched at run time from Harvard Dataverse.

| Field | Value |
|-------|--------|
| Paper DOI | [10.1017/S0003055426101622](https://doi.org/10.1017/S0003055426101622) |
| Dataverse DOI | [10.7910/DVN/BZOCDJ](https://doi.org/10.7910/DVN/BZOCDJ) |
| Manifest | `manifest/dataverse_files.csv` |
| Local cache | `outputs/deposit/` (gitignored) |

Author scripts (`scripts/recodes_*.R`) run **in place** from the deposit cache — not copied into `code/`.
